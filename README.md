# language-list
## foundation model availability for training
Due to the availability of CoNLL-U annotated corpora, the following languages are being modeled: Albanian, Armenian, Basque, Bavarian, Brazilian Portuguese, Bulgarian, Catalan, Croatian, Czech, Galician, Georgian, Haitian Creole, Hausa, Hungarian, Indonesian, Irish Gaelic, Kyrgyz, Naija, Polish, Romanian, Serbian, Scottish Gaelic, Sindh, Slovakian, Slovenian, Spanish, Tamil, Telegu, Turkish, Ukrainian, Uyghur, Vietnamese, Welsh, Wolof
## NER supported languages - derived from Universal NER
Due to the availability of CoNLL-U annotated NER corpora, the following languages are being modeled for NER: Brazilian Portuguese, Cebuano, Croatian, Romanian, Serbian, Slovak, Tagalog
## NER work in progress - derived from other sources
Due to the availability of other NER corpora, the following languages are being formatted as CoNLL-U corpora to be modeled for NER: Catalan, Hausa, Igbo, Naira, Swahili, Wolof, Yoruba
